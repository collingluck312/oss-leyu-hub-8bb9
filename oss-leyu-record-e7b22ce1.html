▎ 站点定位

oss-leyu.com.cn 是一个对象存储相关的服务入口，目前作为静态资源托管与基础文件分发使用。 页面结构简洁，没有复杂跳转，适合直接引用或嵌入到项目中。 下面将从文件组织、典型使用方式以及常见问题三个角度展开说明。

提示：若你正在寻找资源的基础根目录或需确认存储桶路径，leyu 首页会列出可公开访问的顶层目录索引。 对于授权访问的资源，请根据项目文档配置凭证。
▎ 内容组织特点

📁 目录直读

大多数目录默认开启列表，便于直接查看文件名称与大小，无需额外解析。

🔗 链接稳定

资源路径长期有效，不频繁变动，适合嵌入到静态站点或作为 CDN 回源地址。

🧩 命名清晰

文件前缀包含日期或项目代号，方便按需筛选，避免混乱。

🌐 无强制重定向

直接访问链接不会跳转到无关页面，保持纯粹的资源响应。

▎ 典型使用场景
  • 作为个人或团队项目的静态资源基础路径，引用图片、样式或脚本。
  • 备份与归档：将配置文件、日志快照或版本包上传至对应目录。
  • 轻量文件分享：通过链接直接对外提供临时下载，无需额外页面。
  • API 或配置中指定资源 endpoint，统一管理外部资源。
📌 FAQ 摘要 — 快速了解几个细节

问： 是否需要登录才能看到所有文件？
答： 公开桶内的目录无需登录，私密资源需配合访问密钥。首页 leyu 仅展示公开部分。

问： 文件有大小限制吗？
答： 单文件上限取决于存储配置，通常大文件（如 5GB 以内）均可正常上传与下载。

问： 是否支持 wget/curl 直接拉取？
答： 支持，公开资源可直接用 HTTP GET 获取，返回标准响应头。

如果需要直接进入资源索引页，请使用 leyu 作为起始地址。
